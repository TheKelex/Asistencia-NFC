## Avances

En este apartado se llevara un registro de los avances para la realizacion de un MVP.

### Semana 1 (12 - 19 / Julio / 2026)

#### Domingo (12)

Durante este dia se realizo una investigacion mas profundizada sobre las limitantes del sistema y la realizacion de un prototipo para verificar funcionalidad hecho con Claude.

Los hallazgos hechos de este dia fueron:

- Posibilidad de usar el NFC de un dispositivo movil.
- Prohibicion de usar otro dispositivo movil para poder registrarse.
    - Debido a esta prohibicion se estan buscando stickers NFC para realizar la prueba del MVP y para el funcionamiento en general.
    - Debido a que el dia lunes sera festivo no se podra buscar estos en locales de electronico.
- Se descubrio una herramienta para desplegar la pagina con cifrado HTTPS para realizar las pruebas.

**Resumen:** Este dia fue de ardua investigacion acerca de lo que se busca desarrollar lo que nos permitio esclarecer dudas sobre los limitantes del sistema.

#### Lunes (13)

Durante este dia no se llevo un desarrollo inmediato del proyecto debido a que no contamos con los sitckers NFC para verificar que el prototipo funciona para empezar a desarrollar el MVP, una vez tengamos estos se empezaran a realizar pruebas antes de iniciar el desarrollo para evitar problemas futuros.

#### Martes (14)

En este dia se llevo a cabo la busqueda los chips / stickers NFC en el centro de Neiva, mas especificamente en la zona de venta de electronica del centro de la ciudad, se pregunto por este en diversos locales pero en ninguno se encontro el dispositivo necesario para llevar a cabo el MVP.

Debido a esto se tomo la desicion de comprarlos por internet en una distribuidora de electronicos en Bogota, estos requieren un pedido de minimo $20,000 pesos y el envio cuesta $15,900, se pediran 10 tarjetas NFC configurables mediante la pagina [Electronilab](https://electronilab.co/) se hara un pedido de 10 [Tarjeta NFC de Proximidad RFID 13.56 MHz NTAG216 85.5X54mm](https://electronilab.co/tienda/tarjeta-nfc-de-proximidad-rfid-13-56-mhz-ntag216-85-5x54mm/).

Mientras llegan los dispositivos para hacer la prueba se ira realizando el desarrollo FrontEnd.

#### Miercoles (15)

El dia miercoles se realizo la compra de las tarjetas NFC, este dia no se adelanto mas.

#### Jueves (16)

Este dia las tarjetas llegaron a Neiva y estan esperando para ser repartidas por el distribuidor local, a las 9.20 a.m. se llevo a cabo una reunion para repartir desarrollo FrontEnd en base al MockUp propuesto por la clase anterior, ademas de esto se creo el repositorio para llevar un control de versiones y para el trabajo colaborativo (se creo el archivo [Config.md](./Config.md) en el que estan los comandos para inizialisar el repositorio)
