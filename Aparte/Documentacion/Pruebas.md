## Pruebas

Las pruebas deberan ser ejecutadas en un navegador basado en Chromium, un dispositivo con NFC (no dispositivos IOS) y ejecutandose mediante el protocolo HTTPS para la activacion de este.

### Despliegue Recomendado

Usar el aplicativo: app.netlify.com/drop

Este tiene cifrado HTTPS automaticamente y permite el despliegue durante una hora para realizar pruebas.
