## Restricciones

En este apartado se llevara un control sobre los descubrimientos como prerequisitos o restricciones para tener en cuenta al momento de desarrollar el sistema:

- No es posible usar el sistema NFC en dispositivos IOS
- No es posible usar un dispositivo movil para registrase en el sistema (impedimento de sistema).

### PreRequisitos

- Dispositivo con NFC.
- Protocolo HTTPS de intercambio de datos (sin este no se puede usar el NFC).
- Navegador basado en Chromium (Chrome/Edge/Opera/Samsung Internet/Etc.).
